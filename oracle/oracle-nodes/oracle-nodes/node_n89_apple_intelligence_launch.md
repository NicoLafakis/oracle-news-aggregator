## **Verified Context from Research**

### **Key Features:**
**Live Translation:**
- Real-time translation in Messages, FaceTime, Phone
- Audio translation during calls
- AirPods integration for in-person translation
- On-device processing for privacy

**Visual Intelligence:**
- Onscreen content analysis across apps
- Object, plant, landmark identification
- Information extraction from images
- ChatGPT integration for complex analysis

**Siri 2.0:**
- Context awareness across emails, messages, photos
- Multi-app task execution from single commands
- Text-to-Siri functionality
- Personal context understanding

**Creative Tools:**
- Genmoji: Personalized emoji creation
- Image Playground: Text-to-image generation
- Multiple art styles (oil painting, vector art)
- Integration with Messages and Notes

### **Privacy Architecture:**
- **On-Device Processing**: Most AI runs locally
- **Private Cloud Compute**: Encrypted, non-stored processing
- **Data Policy**: No external data storage or marketing use
- **Transparency**: Clear indication when AI is used

### **Supported Devices:**
- iPhone 15 Pro/Pro Max, iPhone 16 series
- iPads with M1/A17 Pro and later
- Macs with M1 and later
- Apple Vision Pro (select features)
