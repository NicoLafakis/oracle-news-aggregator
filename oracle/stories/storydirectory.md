# Oracle Stories

This directory contains all aggregated news stories organized by date and category.

## Structure

```
stories/
└── YYYY/
    └── MM/
        └── DD/
            ├── index.json           # Daily summary
            ├── science_technology/
            ├── computing_ai/
            ├── geological_events/
            ├── political/
            ├── financial_markets/
            ├── business/
            ├── employment_economy/
            └── policy_regulation/
```

Each story is stored as a JSON file named by its content hash.
